Anaconda3 升级命令：

conda update anaconda

conda install spyder=5.1.5

等号后面是版本号

conda update --all

conda update spyder

Anaconda 中升级Spyder 5.1.5不能直接更新的原因(问题是不兼容)

https://stackoverflow.com/a/69713849/17102661

HExo中NEXT主题使用的数学编辑器渲染开启公式：

mathjax: true

Xcode过大，用不到开发os相关软件，删除多余文件：

rm -f

清理xcode 几个大的模拟器（不影响代码链的运行和命令，这里不加sudo会出现无权限的情况）

sudo rm -rf /Applications/Xcode.app/Contents/Developer/Platforms/AppleTVOS.platform/Library/Developer/CoreSimulator/Profiles/Runtimes/tvOS.simruntime

/Applications/Xcode.app/Contents/Developer/Platforms/WatchOS.platform/Library/Developer/CoreSimulator/Profiles/Runtimes/watchOS.simruntime

/Applications/Xcode.app/Contents/Developer/Platforms/iPhoneOS.platform/Library/Developer/CoreSimulator/Profiles/Runtimes/iOS.simruntime

在Parallels Desktop中kali安装Parallels Tools：

这里是官方文档，实际操作中需要将光盘中的内容复制出来，放到一个新的文件夹比如文档文件夹下，新建一个命名为任意的文件夹， 并将复制出来的内容粘贴进去，并在这个文件夹中运行：'sudo ./install'命令，一直点击next即可。
