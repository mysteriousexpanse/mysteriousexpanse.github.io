###Get_Post
url中‘#’‘？’‘&’的作用：井号在url中代表对浏览器发出指令滚动可视区域至井号后面的内容。问号代表的是分割url与参数，而&号则代表参数之间的分隔作用如：变量1&变量2。
php中的变量命名规则为：$变量名 ，其中变量名区分大小写。
[] 在php语法中为短数组语法，意义与array()相同
关于$_GET ,$_POST两种传参方式的个人理解，get命令是在url中将数组赋给当前变量的命令，用于读取资料。而post是在向某个资源提交数据或者文件，数据被包含在http请求的body中。（补充body知识https://en.wikipedia.org/wiki/HTTP_message_body）。相比get是在可见的状态下传输，post的请求个人认为从不可见的方面更加安全，

include(“flag.php”);

##<?php
include(“flag.php”);

{

{
if (md5($_POST[‘idp’]) === md5($_POST[‘gk’]))
echo $flag;
else
echo “Wrong”;
}
else echo “The values of idp and gk cannot be the same”;
}
##
#可以发现两个变量先要满足不相等（弱）然后md5计算后要强相等，弱相等在数值转换后相等也可继续执行，强相等需要在类型、数值上完全相等，此时可用数组处理变量，而不能用0e处理，0e处理是会被认为是科学记数法从而无法满足不等条件

前置小知识：
模运算：如在一个除法运算中。正整数n，整数a，用a/n，余数为r可得到 a = qn+r（0《r《n）在此过程用a mod n表示余数r，即mod表示前面的整数除以后面的整数所得的余数。
rsa算法简介：https://zh.wikipedia.org/wiki/RSA加密演算法
在数学能力不足的情况下可初步认为实现一个rsa加密过程所需要得知的数据有p q(pq为随机的大质数) n（pq） e(随机数)+python若干已知可搜索代码
所需要的python3库有：gmpy2，math，libnum,crypto
在逆向解题时可以将上述过程逆向求解
